About

Twitter Categorization Bot

Bot is designed to create a representation of a user's Twitter Universe.
Using words, topics, and similarities between the root account and accounts connected, TCB can provide a reasonable assumption of a Twitter user's personality.

We use the Twitter API to gather information from the root user's Twitter account.

Particularly, we are pulling a user's TWEETS, FAVORITES, FOLLOWERS, and ASSOCIATED TOPICS

Our data is stored in a remote MongoDB database.

We use the Python NLTK Library for parsing tweets and categorization.

The Categorization Algorithm takes in saved data and compiles into a hash table.

We then rank words, contexts, and topics associated with the root user and subsequent followers.
Our distance algorithm takes compares rankings to determine proper node positioning.

Cytoscape is used to create the visual representation.
